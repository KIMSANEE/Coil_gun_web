장치 개념 설명



MCU(Microcontroller Unit)란?


- 프로그래밍을 통해 다양한 '제어'나 '연산'작업이 가능한 마이크로컨트롤러를 말함



적외선 센서(IR Sensor)란?


- 적외선을 이용해 온도, 압력, 방사선의 세기 등의 물리량이나 화학량을 감지하여 신호처리가 가능한 전기량으로 변환하는 장치



릴레이(Relay)란?


-> ON,OFF가 있는 일종의 '스위치'로 흔히 낮은 전압/전류를 이용하여 더 높은 전압/전류를 제어하는데 사용됨
